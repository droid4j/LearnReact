1. 进入项目目录

   $ cd LearnReact

2. 初始化仓库

   $ git init

3. 添加到暂存区

   $ git add .

4. 提交到版本区

   $ git commit -m "init"

5. 关联远程仓库

   $ git remote add origin https://github.com/droid4j/LearnReact.git

6. 推送到远程仓库

   $ git push origin master

7. 本地创建分支并切换

   $ git checkout -b dev

8. 将本地分支推送到远程仓库

   $ git push origin dev
